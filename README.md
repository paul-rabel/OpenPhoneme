# OpenPhoneme

## What is this? 
OpenPhoneme is a repository that extracts Open Street Map data and uses ML to suplement the data with phonemes. 

## How do get started?

1. Download a .pbf file from Open Street Map through:

    https://www.geofabrik.de/data/download.html

2. Use osm2pgsql to import your .pbf data into a PostgreSQL/PostGIS database

   https://osm2pgsql.org

4. Clone this repository:
   ```
   git clone https://github.com/paul-rabel/OpenPhoneme.git
   ```
