# OpenPhenome

## What is this? 
OpenPhenome is a repository that extracts Open Street Map data and uses ML to suplement the data with phenomes. 

## How do get started?

1. Download an SQL database from Open Street Map through:

    https://www.geofabrik.de/data/download.html

2. Clone this repository:
   ```
   git clone https://github.com/paul-rabel/OpenPhoneme.git
   ```
